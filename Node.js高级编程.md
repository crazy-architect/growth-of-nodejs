# 第3章 加载模块
### 加载模块方式
* 加载核心模块
  > ```var http = require('http');```
* 加载文件模块
  > ```var myModule = require('../my_modules/my_module');```
* 加载文件夹模块
  > ```var myModule = require('./myModuleDir');```
* 从node_modules文件夹加载
### 缓存模块
> 模块在首次加载时会被缓存起来，这意味着如果模块名能够被解析为相同的文件名，那么每次调用require('myModule)都会确切地返回同一个模块
